This project is an open-source based. Every can use this code or join to development of the project.

## Requirements
You need to install this application:
```
git
nodejs
npm
docker
docker-compose
```


## Local project set up
```
git clone URL api
cd api/auth
npm install
cd ../gate
npm install
cd ..
cp .env.example .env
```

## Running the app

```bash
# local development
$ docker-compose up
```
